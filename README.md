AI Nutrition Recommender System

An AI-powered nutrition recommendation system that provides personalized diet plans and meal suggestions based on user inputs such as age, gender, weight, height, activity level, and health goals. The application uses machine learning techniques and nutritional data analysis to recommend balanced diets, calculate calorie requirements, and promote healthy eating habits. Built with Python, Streamlit, Pandas, and Scikit-learn for an interactive and user-friendly experience.

Key Features:

Personalized diet recommendations
Daily calorie and nutrition calculation
Health goal-based meal planning
User-friendly Streamlit interface
Data-driven nutritional analysis
Interactive dashboard and reports

Tech Stack: Python, Streamlit, Pandas, NumPy, Scikit-learn, Machine Learning, Data Visualization.
